# Constructify Frontend 🚀

The **Constructify frontend**, built with **Angular 16** 🛠️, powers a modular web app for construction project management. It’s intuitive, responsive, and packed with features to boost collaboration and efficiency! 🌟 Connects seamlessly with the backend at `http://localhost:4200/back` for real-time data. 💻

## Features
- **Project & Task Management** 📋: CRUD ops, drag-and-drop Kanban, FullCalendar for task visualization with color-coded priorities.
- **Dashboard** 📊: Real-time metrics, Chart.js pie charts, mobile-friendly.
- **Timesheet** ⏰: Log hours, export to PDF/Excel.
- **Deliverables** 📦: CRUD, status tracking, PDF export.
- **Resources** 🧑‍🔧: Manage human/material resources, SMS alerts.
- **Docs** 📄: File/folder CRUD, in-app editor, auto-save.
- **User Auth** 🔐: JWT-based login, email verification, password reset.
- **Issues & Risks** ⚠️: CRUD, notifications, dashboard stats.
- **Finance** 💰: Budget tracking, ROI calc, cost alerts.
- **Communication** 💬: Messages with emoji reactions, AI summaries.
- **Teams** 👥: CRUD, AI name suggestions, secure setup.
- **Reports** 📈: Multilingual, QR codes, advanced search.

## Tech Stack
- **Angular 16.2.16** 🎨
- **Chart.js**, **FullCalendar** 📅
- Responsive for all devices 📱💻
- Secure with role-based auth 🔒
